Poke-Packer

This is a simulation of the mobile game Pokemon TCG Pocket. I created it to challenge myself to learn full stack development using React, Node, Express, and MongoDB.

Note: The website does take around 1 minute to initially load before being able to run the simulation. This is because it is hosted on a free plan on Render.

Console-Based MP3 Player

This project was created by myself and two of my other classmates Luke Alkema and Dylan Shouldice. For our final project in our Data Structures course, we were tasked with solving a real world problem using data structures and algorithms in C to create an application.

So what we designed was a music player that uses text files to persist data as this was before we learned MYSQL databases and programmatic access. I am really proud of how this project turned out and how much I learned from completing this project.
